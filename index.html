Set Table

What is this?

I decided to create a small app that mimicked a code challenge that I did recently. The idea is to be able to enter the number of foods that each person would like, then "set the table" with a button.

How did you make it?

Like most of my projects, I used Bootstrap, Sass, jQuery, Gulp, Bower, and NPM.
